# Heart Kit Demo

A front-end web app for Heart Kit demonstrator.

![](./docs/assets/heart-kit-app.png)


## Installing

`npm i`

## Building

`npm run build`


__Enviroment Variables__

* `REACT_APP_REST_URL`: e.g. `http://localhost:300/api/v1`

## Serving

`npx -y serve -p 80 ./build`

## Development

### Tech Stack

* React
* Typescript
* Mobx-state-tree
* MUI w/ Emotion

## Running

`npm start`
